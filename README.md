# dotfiles and wallpapers for Void Linux
## Features
- Operating_System: Void Linux (glibc)
- Wayland_Compositor: swayfx w/ autotiling-rs
- Status_Bar: swaybar + i3status-rust
- Launcher: fuzzel
- Shell: fish
- Terminal: foot (footclient/server)
